HTML (Hypertext Markup Language): გამოიყენება ვებ-გვერდის სტრუქტურის შესაქმნელად. ის აღწერს ვებ-გვერდის შინაარსს, როგორიცაა ტექსტი, სურათები, ბმულები, ფორმები და სხვა ელემენტები. HTML არ მოიცავს დიზაინს და ფუნქციონალობას, მხოლოდ სტრუქტურას. CSS (Cascading Style Sheets): გამოიყენება ვებ-გვერდის ვიზუალური სტილისთვის. CSS განსაზღვრავს ფერს, შრიფტს, განლაგებას, ზომებს და სხვა ვიზუალურ მახასიათებლებს. ის ვებ-გვერდის ელემენტებს შთაბეჭდილებას აძლევს და მათ უფრო სილამაზეს, ფუნქციონალურობას ანიჭებს. JavaScript: გამოიყენება ვებ-გვერდის დინამიური ფუნქციონალობისთვის. JavaScript ეხმარება ვებ-გვერდებს რეაგირებაში მომხმარებლის ქმედებებზე, როგორიცაა ღილაკზე დაწკაპუნება, ფორმის შევსება, ან სხვა ინტერაქტიული მოვლენები. ის შეიძლება გამოიყენებოდეს მონაცემთა დამუშავებისთვის, შეცვლით ან დამატებით შინაარსზე, გვერდის განახლების გარეშე.
